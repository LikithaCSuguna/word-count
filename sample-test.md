# Testing Word Counter Pro AI

Welcome to your brand new VS Code extension!

This is a sample document created specifically to test the capabilities of the Word Counter Pro AI extension. As you are reading this, or if you start typing new sentences here, the extension should be actively counting the words in the background.

## Check Your Status Bar
Look at the bottom-left corner of your VS Code window right now. You should see a little book icon followed by the word count for this document. If you add or delete words, that number will update automatically!

## Try the Analytics Panel
Want to see more details? Try running the extension's command:
1. Press `Ctrl+Shift+P` (Windows/Linux) or `Cmd+Shift+P` (Mac).
2. Type `Word Counter: Show Analytics` and press Enter.

Alternatively, you can just use the keyboard shortcut `Ctrl+Shift+A`.

You should see a full breakdown of the characters, words, lines, and estimated reading time. Congratulations on building and installing your own production-quality extension!
